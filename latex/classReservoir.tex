\doxysubsection{Detailed Description}
Represents a reservoir vertex in a graph. 

This class inherits from the \mbox{\hyperlink{classVertex}{Vertex}} class and represents a station vertex in a graph. It stores information such as the resrvoir\textquotesingle{}s name, code, max\+Delivery, municipality, and ID.
