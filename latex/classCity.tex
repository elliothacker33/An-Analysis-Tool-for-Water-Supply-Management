\doxysubsection{Detailed Description}
Represents a city vertex in a graph. 

This class inherits from the \mbox{\hyperlink{classVertex}{Vertex}} class and represents a city vertex in a graph. It stores information such as the city\textquotesingle{}s name, code, demand, population, and ID.
